Dev Proxy MockGeneratorPlugin Demo
This app demonstrates how to automatically generate mock files from real API traffic. Make API requests while Dev Proxy is recording, then stop recording to generate mocks.

Workflow: Record Once, Mock Forever

  1. Start Dev Proxy with recording enabled (it's already configured in .devproxy/devproxyrc.json)
  2. Click the buttons below to make API requests through Dev Proxy
  3. Press S in the Dev Proxy terminal to stop recording
  4. Find your generated mocks in mocks-yyyyMMddHHmmss.json
