# Assets

This folder should contain the following assets:

## screenshot.png

A screenshot showing the Dev Proxy CI/CD API validation in action.

**Recommended specifications:**
- Resolution: 1920x1080
- Content: Show a GitHub Actions or Azure Pipelines run with Dev Proxy reports

**Suggested screenshot content:**
- GitHub Actions workflow job summary displaying Dev Proxy validation reports
- Terminal output showing shadow API detection or permission validation results
- Generated OpenAPI spec or markdown reports

To create the screenshot:
1. Set up the sample in a GitHub repository with API Center configured
2. Run the GitHub Actions workflow
3. Capture the workflow summary page showing the generated reports
